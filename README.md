# Linux-command
Learning linux
Overview

This project demonstrates the use of basic Linux commands. It includes examples of creating and renaming files, viewing file contents, searching for patterns, compressing and extracting files, downloading files from the internet, changing file permissions, and working with environment variables.

Topics Covered
1. Creating and Renaming Files/Directories

mkdir – used to create a new directory.

touch – used to create an empty file.

mv – used to rename or move files and directories.

2. Viewing File Contents

cat – displays the entire content of a file.

head – displays the first few lines of a file.

tail – displays the last few lines of a file.

3. Searching for Patterns

grep – used to search for specific words or patterns in a file.

4. Zipping and Unzipping Files

zip – compresses files or directories.

unzip – extracts files from a zip archive.

5. Downloading Files

wget – downloads files from the internet using a URL.

6. Changing File Permissions

chmod – changes file access permissions such as read, write, and execute.

7. Working with Environment Variables

export – sets environment variables in the Linux shell.

echo – displays the value of a variable.

Conclusion

This assignment provides hands-on practice with essential Linux commands that are commonly used in system administration and development environments.
